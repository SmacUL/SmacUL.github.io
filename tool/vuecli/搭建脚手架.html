# 安装 Node.js

# 安装 cnpm

还是网络的问题, 安装一个镜像

# -g 表示全局安装
npm install -g cnpm --registry=https://registry.npm.taobao.org
1
2

安装成功之后, 可以直接使用 cnpm 代替 npm

# 安装 vue-cli

# 直接用 cnpm 也没问题
npm install -g vue-cli
1
2

# 项目构建与测试

# 初始化项目

vue init webpack [项目名]
1

TIP

项目名建议小写. 这个操作结束之后, 将在当前路径下生成一个名称为[项目名]的文件夹.

例如图中, 在 front 目录下查看, 将发现一个 front 文件夹.

# 安装依赖

cd [项目名]
npm install
1
2

# 测试运行

npm run dev
1

正常运行后将报出访问地址.
