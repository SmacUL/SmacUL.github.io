## 目标元素是否滑到底部的判断

 

```
元素的尺寸 = 元素竖直方向的移动长度 + 浏览器显示尺寸 - 其他元素长度
即: 元素竖直方向的移动长度 = 元素的尺寸 - 浏览器显示尺寸 + 其他元素长度
 
所以先设置鼠标滑动的监听器
window.addEventListener('scroll', this.scrollHandler, false);
 
```
