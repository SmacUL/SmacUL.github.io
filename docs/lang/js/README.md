# JS 笔记

因为 JS 和 ECMAScript(ES) 之间紧密而又微妙的关系, 因此这一部分的笔记包括 ES6 内容;
为了方便起见, 笔记中的 JS 指的是当前的版本, ES6 指的是即将发布的新的 JS 标准. 笔记主要以 JS 为主, ES6 作为补充信息;
