# 限制作用域

如果使用模块, 我们比较希望, 模块内的 CSS 只作用在当前模块, 可以在 style 标签上添加 scoped 限制当前 CSS 的作用域;

<style scoped></style>
1

# 影响

但是这个操作会产生点影响, 当前 CSS 不接受上级 CSS 的影响, 同时也无法影响下一级组件的 CSS ;

# 解决

使用 >>> 就可以强制影响下一级组件的 CSS ;

/* 假设当前有类 dad , 他的下一级组件中有类 son */
.data >>> .son {}
